To configure username: git config --global user.name "New Name"
To configure email: git config --global user.email "New Email"
To check git account settings: git config --list
To check one item from the settings list: git config item.name

pwd: print working directory
cd: change directory
~: home

git init: Initializes a Git repository
git add . : add all the changes made to current (.) directory
git add file.txt: add file.txt in the "staging area", it goes from the working copy to the staging area. When a commit command is typed, then it goes into the repository
git commit -m "Message": creates sort of screenshot of current state of files
git log: gives history of commits
git log --author="FullName or PartOfName of author": shows commits of a specific author

git status: compares the working directory to the repository (mumbo jumbo)
